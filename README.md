# Atlas Design & Build website

This folder is the whole website. Every page is a plain HTML file.

- `index.html` is the home page
- `about/`, `services/`, `guides/` etc. are the other pages
- `assets/` holds the styles, fonts and images

To make changes, open this folder with Claude Code and ask it in plain English.
